# Experimental Skills

This directory contains experimental skills that are under active development.

## Before Using

**Always confirm with the user before activating any experimental skill.** These skills:

- May have incomplete functionality
- May produce unexpected behavior
- Are subject to breaking changes without notice
- Should not be used in production workflows without explicit user approval
