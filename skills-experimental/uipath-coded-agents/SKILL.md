# UiPath Coded Agents (Experimental)

> **This skill collection is experimental.** Always confirm with the user before using any sub-skill.

End-to-end toolkit for building, running, and deploying UiPath coded agents.

## Sub-Skills

| Skill | Description |
|-------|-------------|
| `auth` | Authenticate with UiPath Cloud (OAuth or client credentials) |
| `setup` | Scaffold and initialize agent projects |
| `build` | Implement agent logic with framework-specific patterns |
| `run` | Run agents locally or invoke published agents in cloud |
| `evaluate` | Create and run evaluations with built-in evaluators |
| `deploy` | Package and publish agents to Orchestrator |
| `sync` | Push/pull project files to/from Studio Web |
| `bindings` | Framework bindings and integrations |
| `uipath` | Full lifecycle orchestrator (runs all stages end-to-end) |
