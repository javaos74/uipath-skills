   ```bash
   servo targets
   ```
   "Note which windows (w-refs and titles) are already present."

   Step 2 — Run the workflow:
   ```bash
   uip rpa run-file --file-path "<FILE>" --project-dir "<PROJECT_DIR>" --command StartDebugging --format json
   ```

   Step 3 — When done (success or failure) — stop the debug session:
   ```bash
   uip rpa run-file --file-path "<FILE>" --project-dir "<PROJECT_DIR>" --command Stop --format json
   ```

   Step 4 — List windows again:
   ```bash
   servo targets
   ```

   Step 5 — Diff before vs after. Any window present now that was NOT in the baseline was opened by the workflow. Close it:
   ```bash
   servo window <w-ref> Close
   ```

5. **Consequence warning** (from coded line 203):
